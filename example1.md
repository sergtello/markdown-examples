Example 1: Basic Formatting
===========================

This is an example of basic Markdown formatting. Here are some examples of the formatting options available in Markdown:

## Headers

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

## Emphasis

*Italic*
**Bold**
***Bold and Italic***

## Lists

### Unordered List

- Item 1
- Item 2
- Item 3

### Ordered List

1. Item 1
2. Item 2
3. Item 3
